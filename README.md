# Eclipse-UML-Visualizer

An Eclipse plug-in project for visualizing UML diagrams.

## Setup Instructions

1. **Install Eclipse for RCP and RAP Developers**  
   Download from: [https://www.eclipse.org/downloads/](https://www.eclipse.org/downloads/)

2. **Verify PDE is Installed**  
   Go to `Help → Install New Software...`  
   In **Work with:** select your Eclipse release update site (e.g., `https://download.eclipse.org/releases/latest`).  
   Under **General Purpose Tools**, ensure **Eclipse Plug-in Development Environment (PDE)** is installed.

3. **Run the Plugin**  
   Right-click the project → **Run As → Eclipse Application**.  
   A new Eclipse instance will launch with the plugin loaded.
